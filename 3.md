Title
Add AKS Backup Policy, Backup Instance & Configuration Locals
Overview
This PR introduces Azure Data Protection–based backup enablement for the AKS cluster using Terraform.
Components Added
1️ aks_backup_policy.tf
. Creates an Azure Data Protection backup policy with:
. Backup trigger frequency: Every 4 days
. Retention period: 30 days
. Applied to Kubernetes cluster workload
2️ aks_backup_instance.tf
. Creates the backup instance for the AKS cluster.
. Links AKS cluster to Backup Vault
. Associates the 4-day backup policy
. Enables full cluster backup (all Kubernetes workloads)
3️ aks_backup_locals.tf
. Centralized configuration file used to:
. Store reusable backup variables
. Define policy parameters
. Standardize naming conventions
