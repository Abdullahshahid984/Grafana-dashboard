**Title:** feat(istio-gateway): add custom domain support for workloads

**Problem**

The shared Istio Gateway only accepted auto-generated hostnames. There was no way to add a custom domain like `catools-dev.breadfinancial.com` without a manual kubectl patch that gets overwritten on the next Terraform apply.

**Changes**

`workload-onboarding/phases/02-per-instance-per-cluster/locals.tf` - Updated `istio_gateway_hosts` to merge auto-generated and custom hosts using `concat()`. Uses `try()` for backward compatibility with workloads that do not declare `extra_gateway_hosts`.

`bfhaksplatform/istio-ingress-gateway/cicd/install.sh` - Updated cert-manager certificate block to append `extra_sans` to `dnsNames`.
