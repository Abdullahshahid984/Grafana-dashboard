### Step 1 — Create Certs Folder

```powershell
New-Item -ItemType Directory -Path "C:\certs" -Force
```

### Step 2 — Copy Downloaded Files

```powershell
Copy-Item "$env:USERPROFILE\Downloads\6159-provisioned-tenant-ca.pem" "C:\certs\"
Copy-Item "$env:USERPROFILE\Downloads\netskope chain.p7b" "C:\certs\"
```

### Step 3 — Generate Combined PEM from Windows Store

```powershell
((((gci Cert:\CurrentUser\Root) + (gci Cert:\LocalMachine\Root) + (gci Cert:\CurrentUser\CA) + (gci Cert:\LocalMachine\Root)) | Where-Object { $_.RawData -ne $null } `
| Sort-Object -Property Thumbprint -Unique `
| % { "-----BEGIN CERTIFICATE-----", [System.Convert]::ToBase64String($_.RawData, "InsertLineBreaks"), "-----END CERTIFICATE-----", "" }) `
-replace "`r","") -join "`n" `
| Out-File -Encoding ascii "C:\certs\nscacert_combined.pem" -NoNewline
```

### Step 4 — Set Environment Variables (User Level)

```powershell
[System.Environment]::SetEnvironmentVariable("REQUESTS_CA_BUNDLE", "C:\certs\nscacert_combined.pem", "User")
[System.Environment]::SetEnvironmentVariable("AWS_CA_BUNDLE", "C:\certs\nscacert_combined.pem", "User")
[System.Environment]::SetEnvironmentVariable("NODE_EXTRA_CA_CERTS", "C:\certs\nscacert_combined.pem", "User")
```

| Variable | Used By |
|---|---|
| `REQUESTS_CA_BUNDLE` | Python, Azure CLI, Terraform |
| `AWS_CA_BUNDLE` | AWS CLI |
| `NODE_EXTRA_CA_CERTS` | Node.js, npm |

### Step 5 — Git

```powershell
git config --global http.sslCAInfo "C:/certs/nscacert_combined.pem"
```

### Step 6 — pip

```powershell
pip config set global.cert "C:\certs\nscacert_combined.pem"
```

### Done

```powershell
Write-Host "Done! Restart your terminal." -ForegroundColor Green
```

>  **Restart your terminal** after running all steps for environment variables to take effect.

## Verification

Run these after restarting your terminal to confirm everything is set correctly:

```powershell
# Check env vars
[System.Environment]::GetEnvironmentVariable("REQUESTS_CA_BUNDLE", "User")
[System.Environment]::GetEnvironmentVariable("NODE_EXTRA_CA_CERTS", "User")

# Check git
git config --global http.sslCAInfo

# Check pip
pip config list

# Check combined PEM was created
ls C:\certs\
```

### Expected Output

| Check | Expected Value |
|---|---|
| `REQUESTS_CA_BUNDLE` | `C:\certs\nscacert_combined.pem` |
| `NODE_EXTRA_CA_CERTS` | `C:\certs\nscacert_combined.pem` |
| `http.sslCAInfo` | `C:/certs/nscacert_combined.pem` |
| `C:\certs\` | Shows both `.pem` and `.p7b` files |
